Create an account

Use this service to create a new My NHS Pension account.

The account creation process takes around 15 minutes, during which you will be asked to confirm a number of key items of personal information. This information is used to validate your identity against the information that is held on our system.

Before you start

You'll need:

  • your registration code
  • your personal identification code
  • your pension SD reference number
  • your National Insurance number
How to find your information

Registration code

A 6 character code found on the document attached to your invitation email, for example AHJIRU

Personal identification code

An alphanumeric code found on the document attached to your invitation letter, for example 0ejkUDH

Pension SD reference number

An 8 digit number found in the filename of the document attached to your invitation letter, for example 78364736

National Insurance number

You may be able to find it on your:

  • payslip
  • P60
  • letters about tax, pensions or benefits

For more information visit the Find a lost National Insurance number page.

Information:

Once you have created your account you will need to set up two-factor authentication to be able to sign in. You can do this either using an SMS message or an authenticator app on your smartphone.
